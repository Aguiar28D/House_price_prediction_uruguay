# House price prediction in Uruguay

Machine learning project to estimate house prices in Uruguay
using real estate listings data (Mercado Libre + El Gallito).

- Dataset: +37k observations (2023–2025)
- Models: Linear Regression, CatBoost, CatBoost + Optuna
- Best result (Montevideo): MAPE ≈ 25%

> Academic project for exploratory purposes.
